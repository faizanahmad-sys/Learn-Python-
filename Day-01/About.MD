# Python Basics Practice

## Introduction

This project contains basic Python programs and exercises created while learning Python programming. It covers fundamental concepts such as variables, data types, operators, type conversion, user input, and simple problem-solving tasks.

## Topics Covered

### 1. Print Statements

* Displaying text on the screen using `print()`.
* Printing personal information.

### 2. Variables and Data Types

* String (`str`)
* Integer (`int`)
* Float (`float`)
* Using `type()` to check data types.

### 3. Arithmetic Operations

* Addition (`+`)
* Subtraction (`-`)
* Multiplication (`*`)
* Division (`/`)
* Modulus (`%`)
* Exponentiation (`**`)
* Floor Division (`//`)

### 4. Relational Operators

* Equal to (`==`)
* Not equal to (`!=`)
* Greater than (`>`)
* Less than (`<`)
* Greater than or equal to (`>=`)
* Less than or equal to (`<=`)

### 5. Logical Operators

* `and`
* `or`
* `not`

### 6. Assignment Operators

* `=`
* `+=`
* `-=`
* `*=`
* `/=`
* `%=`
* `**=`
* `//=`

### 7. Type Conversion

#### Implicit Type Conversion

Python automatically converts data types when needed.

#### Explicit Type Conversion (Type Casting)

* `int()`
* `float()`
* `str()`

### 8. User Input

* Taking input using `input()`
* Converting input values into integers and floats.

## Practice Tasks

### Task 1: Basic Calculator

Takes two numbers from the user and calculates:

* Sum
* Difference
* Product

### Task 2: Area of a Circle

Calculates the area of a circle using:

Area = π × r²

### Task 3: Even or Odd Checker

Checks whether a number is even or odd.

### Task 4: String Length Counter

Counts the number of characters in a user-entered string.

## How to Run

1. Install Python 3.
2. Save the code in a file named `main.py`.
3. Open a terminal in the project folder.
4. Run:

```bash
python main.py
```
