## 📌 What is a Function?
A function is a **block of statements** designed to perform a specific, isolated task. Instead of writing the same code repeatedly, we wrap it inside a function and call it whenever needed.

### 💡 Key Characteristics
* **Reusability:** Write once, use multiple times.
* **Modularity:** Breaks complex code into smaller, manageable chunks.
* **Readability:** Makes the overall codebase clean and professional.
