I am faizane data inside variables while running a Python program, that data is stored in volatile memory (RAM) and gets deleted as soon as the program terminates. To store data **permanently**, we use **Files**.

Python supports two main categories of files:
1. **Text Files:** `.txt`, `.py`, `.java`, `.html`, etc. (Structured as human-readable characters).
2. **Binary Files:** `.mp4`, `.png`, `.jpg`, `.pdf`, etc. (Structured as raw machine-readable bytes).
